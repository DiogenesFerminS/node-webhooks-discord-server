# LOGGER DE GITHUB

1. Clone the repository to your local machine
2. Install dependencies by running: ```npm install ```
3. Configure GitHub Webhook: Add a webhook to your GitHub repository, providing your server's endpoint and an Auth Secret.
4. Configure Discord Webhook: Create a webhook in your Discord server and save the URL for your environment variables.
5. Set up Environment Variables: Rename .env-template to .env and fill in the required values.
6. Build the project: ```npm run build``` 
7. Start the application: ```npm start```
