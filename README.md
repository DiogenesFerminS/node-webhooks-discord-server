# LOGGER DE GITHUB

1. CLONA EL REPOSITORIO
2. RECONSTRUYE LOS MODULOS DE NODE ```npm install ```
3. PRIMERO AÑADE UN WEBHOOK A TU REPOSITORIO DE GITHUB Y ASIGNALE EL ENPOINT ADEMAS DEL SECRET DE AUTH
4. AÑADE UN WEBHOOK EN TU SERVER DE DISCORD Y GUARDA SU URL PARA ASIGNARLA EN LAS ENVS
5. RENOMBRA EL .env-template A .env Y ASIGNA LAS VARIABLES 
6. ```npm run build``` 
7. ```npm start```
